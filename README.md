# Shape of Triangle

## 問題

以下のように、与えられた三角形の3辺の長さから、三角形の形を表示するプログラムを作成してください。この時、与えられる辺の長さは整数とします。また、コマンドライン引数はカンマ区切りで辺の長さを与えるようにしてください。

````
$ ruby triangle.rb 2, 3, 4
不等辺三角形ですね！

$ ruby triangle.rb 2, 2, 1
二等辺三角形ですね！

$ ruby triangle.rb 1, 1, 1
正三角形ですね！

$ ruby triangle.rb 1, 2, 3
三角形じゃないです＞＜
````
